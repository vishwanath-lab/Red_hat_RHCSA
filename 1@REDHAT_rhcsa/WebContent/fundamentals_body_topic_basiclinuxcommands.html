Commands are case sensitive

  • pwd - present working directory
  • ls - list , -l - long list, -a - hidden list
  • cd - change the current directory, cd..
  • more or cat - display files
  • clear - clear the screen
  • man - manual page, man -k diff
  • echo - display arguments in the screen $echo $PATH
  • exit - exit the screen
  • which - locate the command

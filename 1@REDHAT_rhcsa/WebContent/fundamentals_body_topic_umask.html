Umask - User mask

New files are created with a default set of permissions. Specifically, a new file's permissions may be restricted in a specific way by applying a permissions "mask" called the umask.

#umask - to know current umask set
0022

Masking or hiding those permission we don't want to default to have.
By default without umask file permission is 666 and directory is 777

If we set mask 0022 means file permission should be 666-022 = 644 (rw-r-r)

Directory 777-022 =755 = rwx-rx-rx

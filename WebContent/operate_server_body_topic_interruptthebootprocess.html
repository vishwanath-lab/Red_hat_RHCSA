Recover root password :-

  1. Reboot the system
  2. Go to grub menu and add the the end rd.break.. And press ctrl+x ..this will boot to the emergency mode
  3. Mount the /sysroot as rw
  4. Do the chroot.. Chroot as act as a root director..this will provide access to passwd command
    #chroot /sysroot
  5. Change the root passwd
    #passwd Enter new password :-
  6. Relabel for selinux to recoganize
    #touch /.autorelabel
    #exit
    #exit

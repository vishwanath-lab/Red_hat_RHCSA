Directory - just like the folder
File - contains data

cd . - current directory
cd .. - parent directory
cd - change to the previous directory

File :-

Mostly all use vi or nano or vim

#vi test.txt
i - insert text
:q! --> quit immediately without save
:wq! --> save quickly with save
Esc --> escape from
yy --> copy
p - past below the cursor
U - undo
P - past above my cursor
dd - cut
5dd - 5 line from cursor
5yy - past all the 5 line
Shift+G - go to bottom
1+Shift+G - Go to the first line
cc - remove entire line and change to insert mode
/search - search pattern
R - replace mode
Replace the word with other word

Current target :-

# systemctl get-default

4 common targets :-
  1. Multi user target - multi user login.. Command text based
  2. Graphical.target - GUI based ( # systemctl list-dependencies graphical.target)
  3. Emergency.target - This will boot the system and mount root file system as read only file system.
  4. Rescue.target - Boot with single user environment with minimal loaded for troubleshooting


Move the target manually :-

#systemctl isolate multi-user.target ==> This will switch to multi-user target
#systemctl set-default multi-user.target ==> set the target permanently..need to reboot after change the new target.

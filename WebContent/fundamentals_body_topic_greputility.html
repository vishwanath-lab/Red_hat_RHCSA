Allows us to search specific pattern. We will use grep for searching pattern in the log files and analyzing.
  1. Grep only start with commented items
    # grep '^#' /etc/profile
  2. Grep the uid without case sensitive
    grep -i 'uid' /etc/profile
  3. Search root user available
    [root@susi etc]# grep root /etc/passwd
    root:x:0:0:root:/root:/bin/bash
  4. Ignore commented line and print remaining
    [root@susi etc]# grep -v '^#' /etc/profile
    -v, --invert-match
  5. Grep pattern in /var/log/secure file
  6. Grep all the line end with RX ..
    $ - match the ending position on the string
    [root@susi etc]# grep 'RX$' /var/log/messages
    Aug 13 09:05:39 susi kernel: e1000: enp0s3 NIC Link is Up 1000 Mbps Full Duplex, Flow Control: RX
    Aug 13 09:05:39 susi kernel: e1000: enp0s8 NIC Link is Up 1000 Mbps Full Duplex, Flow Control: RX
    Aug 13 09:08:53 susi kernel: e1000: enp0s3 NIC Link is Up 1000 Mbps Full Duplex, Flow Control: RX
  7. Grep all the pattern not match with particular pattern.
    [root@susi etc]# grep '[^RX]' /var/log/messages

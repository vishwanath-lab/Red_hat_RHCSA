Link to other file on the system

Symbolic link :-

Like windows short link.
Symbolic link can create across the file system.
Symb link have full permission but ultimatly this have only target permission and will not provide full permission to edit.

#ln -s taget link_name
#ln -s /etc/motd motd1
#rm motd1 --> this won't harm the original file
#rm /etc/motd --> this will broke the motd link file

Hard link :-

Link to specific inode location on our file system..Hard link can be created inside the same file system.

If you change the permission to one file and also automatically change to other file.
Remove original file wont affect hard link file because it link to the inode.

#ln motd motd1
